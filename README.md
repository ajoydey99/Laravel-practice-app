# Laravel Practice App 

This is a **practice-based Laravel project** where I explore and implement core and advanced Laravel concepts step by step.

> ⚠️ This project is a work in progress and continuously evolving.

---

## 📌 Features Implemented

* Custom Authentication System (without Laravel Breeze/Jetstream)
* CRUD Operations (Create, Read, Update, Delete)
* Form Validation using Request Classes
* Image Upload Handling
* Pagination
* Flash Messages (Session-based)

## ⚙️ Installation

Follow these steps to run the project locally:

```bash
git clone https://github.com/your-username/laravel-practice-app.git
cd laravel-concepts-lab

composer install
cp .env.example .env
php artisan key:generate
```

### Configure Database

Update your `.env` file:

```
DB_DATABASE=your_db_name
DB_USERNAME=root
DB_PASSWORD=
```

Run migrations:

```bash
php artisan migrate
```

---

## ▶️ Run the Application

```bash
php artisan serve
```

Visit:

```
http://127.0.0.1:8000
```
